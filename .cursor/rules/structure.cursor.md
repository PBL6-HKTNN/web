## Project structure

All development files are stored under `src`

With the TypeScript alias defined, the project structures are detailed below:

- `@components`: components defining
- `@conf`: configuration data, including constant defining
- `@contexts`: React contexts defining
- `@hooks`: React custom hooks defining, also tanstack query hook custom defining in `@hooks/queries`
- `@lib`: Generated lib by shadcn
- `@routes`: Defined routes/pages are stored here using `@tanstack/react-router`
- `@services`: Api service defining
- `@types`: TypeScript type defining
- `@utils`: Helper/utility functions will be defined here
